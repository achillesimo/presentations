Les web components

Les composants web permettent aux développeurs de créer des balises HTML personnalisées et réutilisables.

Les composants web s'appuient sur :

  • Custom Elements: pour créer et enregistrer des balises HTML

  • Shadow DOM : attributs public ou privé des éléments,
  • HTML imports: CSS, JavaScript, ...

  • Templates : squelettes pour des éléments HTML instanciables
